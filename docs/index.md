# 欢迎使用 StarryBot！

StarryBot 是一款可在 QQ 中使用的机器人，功能以音游 Cytus II 为主。也有其他好玩的东西！

这份文档将指引你如何使用 StarryBot。

## 添加 Bot

目前有两个正式运营的公开账号：

- starrybot#0919 (一号机)

- starrybot#0920 (二号机)

它们的功能完全一致。
