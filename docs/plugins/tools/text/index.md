# 文本处理

主命令：`text`

[[toc]]

## 罗马数字转换

```
roman <数字>
```

命令别名：`罗马数字`

将十进制正整数转换为罗马数字。

## 反转文字

```
reverse <文本内容>
```

命令别名：`文字反转`

## 替换文字

```
replace <被替换文本> <替换后文本> <待替换文本>
```

命令别名：`文字替换`

将三个参数依次称为 A、B、C，此命令的作用是将 C 中的所有 A 都替换为 B。

## Zalgo 文本生成

```
zalgo <文本内容>
```

可用选项：

- `--intensity` `-i` 混乱程度
  - 参数：`<混乱程度>`
  - 默认值：`3`
