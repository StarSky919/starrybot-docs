## 表情包

主命令：`表情生成`

可生成的种类非常多且会持续更新，请在聊天环境使用 `help 表情生成` 查看详细信息。

## Make It A Quote

```
miq [内容]
```

命令别名：`q`

引用一条消息即可生成语录！
