# Cytoid

与 Cytoid 相关的功能。

::: warning

由于 Cytoid 的服务器并不稳定，有时会查询失败。此为正常现象。

:::

---

主命令：`ctd`

[[toc]]

## 计算单曲 Rating

```
ctd.rt <等级> <精准度>
```

## 绑定 Cytoid 账号

```
ctd.bind <Cytoid ID>
```

## 解绑 Cytoid 账号

```
ctd.unbind
```

## 查询 Best 30

```
ctd.b30 [Cytoid ID]
```

如果不指定 `[Cytoid ID]`，会使用已绑定的账号作为查询目标。
